# Mywoods Website

Simple React JSX furniture website with Router, Axios, localStorage, Node.js API, and MongoDB.

## Install

```bash
npm install
```

## Run Frontend

```bash
npm run dev
```

## Run Backend

Create `.env` from `.env.example`, then run:

```bash
npm run dev:server
```

Frontend: `http://localhost:5173`
Backend: `http://localhost:5000`

## API

- `GET /api/woods`
- `POST /api/woods`
- `DELETE /api/woods/:id`
